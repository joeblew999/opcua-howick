# Speckle Server

## Workflow

**First time ever (once):**
```
mise run speckle:server:setup
```

**Every day:**
```
mise run speckle:server:start
```

**When done:**
```
mise run speckle:server:stop
```

**Something broken?**
```
mise run speckle:server:status
mise run speckle:server:test
mise run speckle:server:logs
mise run speckle:server:restart
```

## When running things, make it easy for the developer to see what is happening!
