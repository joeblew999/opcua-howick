## Notes

- Item 3 (power supply) is for the Pi 5. The Pi Zero 2W can use any standard USB phone charger (5V, minimum 1A). Use an extension cable if the nearest power outlet is far from the machine.
- Item 5 (cable) connects the Pi Zero 2W to the Howick FRAMA USB port. 3m is usually enough — measure the distance from the machine to where you plan to place the Pi Zero 2W.
